# Changelog

## [1.0.0] - 2026-02-24

### Added
- Initial release
- Image conversion: PNG, JPG, WEBP, GIF, BMP, TIFF, ICO, HEIC, TGA, PPM
- Document conversion: PDF, DOCX, TXT, CSV, XLSX, XLS, PPTX, RTF, HTML, MD, JSON, XML, ODT
- Audio conversion: MP3, WAV, FLAC, AAC, OGG, M4A, WMA, OPUS, AIFF
- Video conversion: MP4, MOV, AVI, MKV, WEBM, FLV, TS, M4V, 3GP, WMV, MPEG
- Video to GIF and video to MP3 (audio extraction)
- Ebook conversion via Calibre: EPUB, MOBI, AZW3, FB2
- Archive conversion: ZIP, TAR, GZ, 7Z, BZ2
- Dark theme UI with sidebar category navigation
- Animated progress bar
- File size display
- Format chips grouped by type (Lossy/Lossless, Raster/Special, etc.)
